# Telegram-бот "Заявка"

Проверочная задача: бот без LLM, long-polling, белый список, сбор заявки по 5 шагам.

## Запуск

1. Создайте тестового бота через BotFather.
2. Скопируйте `.env.example` в `.env` и заполните значения.
3. Запустите:

```bash
python -m app.bot
```

Зависимостей нет: используется только стандартная библиотека Python.

## Настройки

- `TELEGRAM_BOT_TOKEN` - токен Telegram-бота.
- `TARGET_CHAT_ID` - чат, куда бот отправляет готовую заявку.
- `ADMIN_IDS` - стартовый белый список, через запятую.
- `DATA_DIR` - папка для хранения состояния и списка пользователей.

## Команды

- `/start` - начать работу.
- `/adduser <id>` - добавить пользователя в белый список, только для админов.
- `/users` - показать белый список.

## Проверка

```bash
python -m unittest discover tests
```

## Состав сдачи

- `app/` - чистовая реализация.
- `tests/` - сценарные тесты.
- `review_buggy_copy/` - копия с умышленными ошибками для ревью.
- `INTENTIONAL_BUGS.md` - список умышленных ошибок.
- `REPORT.md` - короткий отчет.
