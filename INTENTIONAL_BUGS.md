# Умышленные ошибки в `review_buggy_copy`

1. Белый список хранится только в памяти и теряется после перезапуска.
2. Сессия заявки общая для всех пользователей, поэтому ответы двух людей могут смешаться.
3. Текст вместо кнопки на шаге бюджета принимается как валидный бюджет.
4. Готовая заявка не отправляется в `TARGET_CHAT_ID`.
5. `/adduser` доступен любому пользователю из белого списка, а не только админам.
